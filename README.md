
  # Portafolio — Aarón Martinez

  Portafolio personal, generado a partir de un diseño de Figma Make y desplegado como sitio estático en GitHub Pages.

  ## Correr en local

  Ejecuta `npm i` para instalar dependencias.

  Ejecuta `npm run dev` para levantar el servidor de desarrollo.

  Ejecuta `npm run build` para generar el sitio de producción en `dist/`.
